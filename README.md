Readify is an online bookstore web application. It lets users browse books, search for titles, filter by category, add books to a shopping cart, save books to a wishlist, and switch between light and dark mode. Everything works in the browser without needing to install anything.

I built this project to show that I understand the core technologies of the web. I did not use React, Angular, Vue, or any other framework. Every line of code is vanilla HTML, CSS, and JavaScript. This proves I know how the web actually works under the hood.
